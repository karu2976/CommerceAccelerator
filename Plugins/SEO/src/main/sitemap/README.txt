1.Install the sitemap generator.It will be available with Tools and frameworks.

2.Run the generateSiteMap.bat/sh and provide the inputs.

3.It will generate the site map xml files into sitemap_store folder which can be configured in conf.xml.

4.Update the below details in the conf_CSAxx.xml for the EAC application you wish to generate a sitemap for.  
  The conf file can be found in the config_store folder of your application.

  hostname of the MDEX : localhost - replace with your MDEX host name.
  port number : MDEX port number.
